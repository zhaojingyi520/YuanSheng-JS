1、将远程仓库克隆到本地：
     git clone https://~~~~
2、以下步骤必须位于仓库内执行（验证是否进入仓库，看是否有蓝色master分支，若没有，进行下面操作就有了）：
     cd 仓库名，
2、查看仓库状态：
      git status
3、将更改内容添加到本地仓库
     git add .或*，表示所有内容都保存，
                 后面也可以跟文件名称或文件夹名称表示只保存该文件内的内容
4、提交备注
     git commit -m "备注内容可以是中文"
5、提交到远程仓库
     git push
